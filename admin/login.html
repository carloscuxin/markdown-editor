Editor de Documentación

Ingresa tu Personal Access Token de GitHub para empezar.

¿Cómo obtener tu token?

  1. Ve a github.com/settings/tokens
  2. Haz clic en Generate new token → Fine-grained token
  3. En Repository access, selecciona Only select repositories y elige carloscuxin/markdown-editor
  4. En Permissions → Contents, selecciona Read and write
  5. Genera el token, cópialo y pégalo aquí
