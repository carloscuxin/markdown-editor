# Release Notes

Release notes record **what shipped, when, and what changed** for each version of the Helios Payments Platform. They are the human-readable companion to the technical changelog and GitHub Releases.

!!! note "How this maps to GitHub"
    On rollout, each of these pages corresponds to a **GitHub Release** tagged in the repo (e.g. `v2.4.0`). GitHub Releases can also carry build artifacts and packages — the equivalent of publishing to JFrog Artifactory.
