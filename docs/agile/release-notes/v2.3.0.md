## Highlights

A refreshed web console and more resilient webhook delivery.

## New features

- Redesigned **payments console** with improved accessibility (WCAG 2.1 AA).
- **Webhook automatic retries** with exponential backoff.

## Improvements

- Faster transaction search (average query time down 55%).
- Expanded audit log detail for compliance exports.

## Fixes

- Corrected timezone handling in scheduled reports.
- Fixed a session timeout that logged users out prematurely.

## Upgrade notes

No action required. Webhook consumers should ensure endpoints are idempotent to handle retries.
