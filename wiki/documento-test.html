Documentación Técnica - Módulo de Autenticación

Resumen:
Este módulo maneja la autenticación y autorización de usuarios
mediante JWT. Soporta roles: admin, editor, viewer.

----------------------------------------------------------------------
Tabla de Endpoints
----------------------------------------------------------------------
| Método | Ruta              | Descripción                | Auth |
|--------|-------------------|----------------------------|------|
| POST   | /api/auth/login   | Inicia sesión              | No   |
| POST   | /api/auth/refresh | Renueva access token       | No   |
| GET    | /api/auth/me      | Perfil del usuario autent. | Sí   |
| POST   | /api/auth/logout  | Cierra sesión              | Sí   |
----------------------------------------------------------------------
